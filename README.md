# 💰 Power BI Para Análise de Dados Financeiros

## 📊 Sobre o Projeto

Dashboard desenvolvido durante meus estudos de Power BI, com foco na análise de dados financeiros.

Neste projeto foram explorados indicadores de receitas, despesas, margem de lucro e análise dos principais segmentos financeiros.

Neste projeto, explorei o uso do amarelo como cor de destaque na composição visual do dashboard.

## 📌 Principais Indicadores

- Total de Receitas: **1,92 Mi**
- Total de Despesas: **1,15 Mi**
- Margem de Lucro: **39,96%**

## 📈 Análises Desenvolvidas

- Total de Receitas por Componente;
- Total de Despesas por Componente;
- Análise dos principais segmentos;
- Tabela de sumário financeiro;
- Indicadores financeiros utilizando DAX;
- Hierarquia de datas;
- Tratamento e transformação dos dados no Power Query.

## 🛠️ Ferramentas

- Power BI
- Power Query
- DAX
- Modelagem de Dados
